TOW00000005	TOW Style Minigames
TOW00000006	Save to INI
TOW00000007	Click to Save
TOW00000008	Saves the current configuration to the mod's INI file. You can find it under `Config\TOW Style Minigames.ini`.
TOW00000009	Use INI File
TOW00000010	Uses the included INI file for all configuration of The Outer Worlds Style Minigames. Disable to instead use the MCM for all configuration.
TOW00000011	Require Manually Saving to INI
TOW00000012	Enabling this setting will require manually saving the current configuration to the mod's INI file. Disable to let the mod automatically save the current configuration to the INI when the MCM closes.
